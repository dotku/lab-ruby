# Ruby

一个日本人发明的编程语言，类似 Basic 的语法，目前 Apple 等公司都有用 Ruby 语言。

## 技术

在 Ruby 环境下一些技术相伴产生，比如 HAML，SLIM 等，甚至现在流行的 Node.js 框架 Express 所使用 Jade 也受其影响。

- [HAML](http://haml.info/)，我最早在 2RedBeans 接触到的 HTML 模板语言。
- [SLIM](http://slim-lang.com/)，一个类似 HAML 的模板语言，比 HAML 要干净一些

**备注**

在 PHP 中，有一款与 SLIM 同名的框架叫做 [Slim Framework](http://www.slimframework.com/)，语法方式很类似 Node.js，用来编写 API 会很方便，有兴趣的朋友，可以去了解一下。
